## 📝 Descrição 

Projeto feito em faculdade com o objetivo de criar uma empresa ficticea e um produto condizente que represente uma solução tangível que contribua com as [17 Metas até 2030](https://brasil.un.org/pt-br/sdgs) da ONU. O tema escolhido foi "Trabalho decente e crescimento econômico" e "Redução das desigualdades".
